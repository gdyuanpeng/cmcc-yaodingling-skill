## 功能详解

### 指示灯状态

| 状态 | 含义 |
|------|------|
| 绿色常亮 | 电量高于20%，已连接APP |
| 红色常亮 | 电量低于20%，需要充电 |
| 闪烁 | 未连接APP/等待配网 |
| 熄灭 | 5分钟无操作自动休眠 |

### 功能键操作

| 操作 | 功能 |
|------|------|
| 短按1秒 | 开机（P100）/ 进入配网模式（P800） |
| 长按3秒 | 进入配网模式（P700） |
| 长按5秒 | 强制解绑设备 |

### 复位孔

当药盒出现异常（卡顿、无法绑定等）时，使用牙签或卡针轻戳复位孔，听到"欢迎使用智能药盒"提示音后，设备重启完成。

## 使用指南

### 一、P700 移动爱家配网流程

#### 1. 准备工作
- 下载"移动爱家"APP
- 注册并登录账号
- 确保手机连接2.4G WiFi（不要选5G）
- 准备药盒，确保有电

#### 2. 绑定设备（三种方式）

**方式一：扫码绑定（推荐）**
1. 打开药盒卡扣
2. 长按功能键3秒，听到"等待连接"提示音
3. 打开移动爱家APP → 设备页 → 右上角"+"
4. 点击"扫一扫"，扫描药盒机身或彩盒上的二维码
5. 对比MAC地址后6位，确认一致后点击"连接"
6. 输入WiFi密码，等待连接成功

**方式二：分类查找**
1. 添加设备 → 健康养老 → 智能药盒
2. 选择"药叮铃智能药盒YP-P700"
3. 按提示完成绑定

**方式三：搜索名称**
1. 添加设备页面搜索"药叮铃"
2. 选择对应型号完成绑定

#### 3. 配置用药计划

**添加药品：**
- 扫码添加：点击"扫码药盒"，对准药盒条码扫描
- 手动输入：在输入框输入药品名称

**设置提醒：**
1. 用药频率：每日几次
2. 服药时间：滑动选择具体时间
3. 用药天数：设定疗程天数
4. 确认下发：点击"确定并下发"，听到"配置成功"即完成

## 常见问题与故障排除

### 绑定问题

**Q：长按功能键没有反应？**
- 先检查电量，低电量时指示灯为红色
- 连接Type-C充电，充满约2小时后再试

**Q：搜不到设备？**
- 确保药盒卡扣已打开（指示灯闪烁）
- 安卓/iOS需对比机身贴纸MAC地址后6位
- 重启药盒或复位后再试

**Q：配网失败？**
- 确认连接的是2.4G WiFi（非5G）
- 检查WiFi密码是否正确
- 将药盒靠近路由器重试

### 使用问题

**Q：解绑后还能听到闹铃？**
- 打开卡扣，指示灯常亮后
- 长按功能键5秒听到"解绑成功"提示音
- 或在APP内重新绑定后再解绑

**Q：闹铃不准时响？**
- 打开APP等待设备自动连接同步时间
- 检查用药计划是否成功下发（需听到"配置成功"）

**Q：语音卡顿？**
- 电量不足，请及时充电
- 充满电后如仍卡顿，尝试复位重启

**Q：设置用药计划失败？**
- 确保卡扣打开，指示灯常亮
- 重新点击"确定并下发"
- 听到"配置成功"才算完成

### 硬件问题

**Q：指示灯不亮？**
- 电池耗尽，请充电
- 充电2小时后重试

**Q：无法充电？**
- 检查Type-C线是否正常
- 更换充电头或数据线测试
- 如仍无法充电，联系售后

## 清洁与维护

1. **日常清洁**：使用柔软干净的布擦拭内仓
2. **防潮注意**：不可弄湿电子仓部分，避免短路
3. **加药前**：确保药盒内仓清洁、干燥
4. **长期存放**：充满电后再存放，防止电池过度放电
5. **充电习惯**：首次使用前请先充满电

---

## 注意事项

1. **安全存放**：将药盒放在安全位置，防止跌落损坏
2. **环境要求**：远离明火和热源

5. **故障处理**：出现故障立即联系客服，不要自行拆卸
